🚀 Adım Adım Yükleme

1GitHub'a giriş yap
github.com adresinde hesabın yoksa ücretsiz oluştur.
2Yeni repo oluştur
Sağ üstteki + → New repository
• Repository name: halisaha
• Visibility: Public ✓ (Pages için zorunlu)
• Create repository'ye bas
3Dosyaları yükle
Repo sayfasında Add file → Upload files
Yukarıdaki 4 dosyayı sürükle-bırak veya seç
Altta Commit changes'e bas
4GitHub Pages'i aktif et
Repo sayfasında Settings → Pages
• Source: Deploy from a branch
• Branch: main → / (root)
Save'e bas
5URL'ni al ve paylaş!
Birkaç dakika sonra uygulaman şu adreste:
https://KULLANICI_ADIN.github.io/halisaha/
Bu URL'yi WhatsApp mesajlarında davet linki olarak kullanacaksın.

⚙️ Önemli Ayar

Uygulamayı GitHub'a yükledikten sonra, Organizasyon sekmesinde maç oluştururken Organizatör WhatsApp Numarası alanını doldur. Oyuncuların cevapları bu numaraya gelecek.

⚠️ Davet linkleri sadece uygulama internette yayında olduğunda çalışır. Localhost'ta test etmek istersen linki elle kopyalayıp aynı tarayıcıda açabilirsin.

🔄 Güncelleme

Yeni versiyon geldiğinde: GitHub repo sayfasında index.html'ye tıkla → kalem ikonuyla düzenle → ya da Add file → Upload files ile üzerine yaz.
